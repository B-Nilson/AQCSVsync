
# AQCSVsync

<!-- badges: start -->
<!-- badges: end -->

The goal of AQCSVsync is to enable syncing of the [AQCSV files produced on the AQmap server](https://aqmap.ca/aqmap/data/AQCSV/) with the ECCC Science Network for supporting internal tool development.

When installed on Sci. Net., allows for pulling the files from AQmap (including replacing old files with updated versions)

## Installation

You can install the development version of AQCSVsync like so:

``` r

# install.packages("pak")
pak::pak("B-Nilson/AQCSVsync")

```
